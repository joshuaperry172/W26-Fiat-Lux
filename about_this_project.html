About Our Project

[ PROJECT OVERVIEW ]

This project was designed for a Fiat Lux Seminar at UCLA in February of 2026. It analyzes the parallels between modern ideas of cyberpunk and the ways they are based on the world we live in. This is done by analyzing similarities between the architecture and infrastructure of LA and that of the Cyberpunk 2077 world.

DEVELOPMENT_CREDITS

This project uses images from the Cyberpunk: Edgerunners TV Series as well as clips from the Cyberpunk 2077 video game. The website was designed on GitHub by Joshua Perry with the aid of open-source code and AI. All materials used are for educational and commentary purposes only.
