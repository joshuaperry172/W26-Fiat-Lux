About Our Project

[ PROJECT OVERVIEW ]

This project was designed for a Fiat Lux Seminar at UCLA in February of 2026 by Joshua, Jiah, Iva, and Finn. It analyzes the parallels between modern ideas of cyberpunk and the ways they are based on the world we live in. This is done by comparing and contrasting the architecture and infrastructure of LA and the Cyberpunk 2077 world. We hope you enjoy exploring our website!

DEVELOPMENT_CREDITS

This project uses images from the Cyberpunk: Edgerunners TV Series as well as clips from the Cyberpunk 2077 video game. The website was designed on GitHub by Joshua Perry with the aid of open-source code and AI. All materials used are for educational and commentary purposes only.

AI_USE_STATEMENT

AI, specifically Google Gemini, was used to design the code formatting this website, including animations, styling, and embedding images and maps. All images, map locations, text, analysis, etc., as well as all planning done in the development of the website, have come from the members of the group; AI was used solely to format and make our website more accessible and interesting.
